# Greenden
![image alt](https://github.com/Jaicodes-ops/Greenden/blob/778e1589f1e93d4439e1c21da00921987e051bd2/images/11471878.png)


Greenden is a simple and elegant plant store website built using HTML, tailwindCSS, and basic frontend concepts. The project focuses on showcasing a wide assortment of indoor and outdoor plants with a clean, user-friendly design.

The website is designed to promote greenery and encourage people to bring nature into their everyday lives. It features a modern layout, responsive sections, and visually pleasing plant cards that make browsing enjoyable.
